A forum plugin for WordPress. Provides support for access control, uploadable avatars, achivements, and more.

== Description ==

A forum plugin that has the following capabilities:

* Group based access control
* Groups may have read/write/start access to forums
* Threads can be pinned and closed by moderators
* Users can have uploaded avatars that gets used across the whole site
* Badges can be given to users for achivements and as rank/informative symbols

== Installation ==

1. Upload the plugin directory to `/wp-content/plugins/`
1. Activate the plugin on the dashboard
1. Select and promote moderators on the plugin admin page
1. Create groups and assign them to the users (optional)
1. Create categories and forums
1. Set permissions on the forums and categories

== Frequently Asked Questions ==

= I gave permission to users to read a forum, but they can't see it, why? =

If the forum is inside a category, then you must give read access to that category too. Otherwise every forum inside it will be hidden.
